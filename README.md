# kerma-inputs
Collection of benchmarks used as inputs for Kerma

The script `gen_compile_db.sh` will create a file `compile_commands.json` for each benchmark.
It does so by using the `bear` tool. To install bear on Ubuntu run:
```
sudo apt update
sudo apt install bear
```
## Single file

- b+tree
- backprop
- bfs
- cfd
- gaussian
- hotspot
- lavaMD
- lud
- nn
- nw
- particlefilter
- pathfinder
